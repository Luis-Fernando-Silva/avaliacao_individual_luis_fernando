Olá, Seja bem vindo ao Restaurante Oriental!

° Primeiramente importe o arquivo através do mavem project.

° Após isso, crie um banco de dados com o nome restaurante_oriental e execute
o package AvaliacaoIndividualApplication através do RUN AS -> JAVA APPLICATION.

° Rodando o código, abra seu postam e comece a realizar as pesquisas!

(OBS: Abra a application properties e certifique de verificar se sua conexão
 com banco de dados está feita corretamente!)
